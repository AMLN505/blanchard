# Blanchard
## О проекте
Верстка лендинга художественной галереи.

## Моя роль в проекте
Frontend-разработчик

## Инструкция по запуску
1. Скачать сборку из ветки main
2. Открыть файл index.html в IDE и запустить расширение LiveServer
